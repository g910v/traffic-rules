Ограничение жилой зоны:
1. Запрещена учебная езда
2. Ограничение скорости 20 км/ч
3. Подача звукового сигнала только для предотвращения ДТП
4. Запрещено сквозное движение (объезжать пробки через дворы нельзя, движение без остановки/стоянки)
5. Стоянка с работающим двигателем запрещена

> *Важно: Выезд с жилой зоны не является перекрестком. Поэтому необходимо уступать дорогу всем ТС, которые осуществляют движение по дороге, пешеходам, велосипедистам и т.д.*
