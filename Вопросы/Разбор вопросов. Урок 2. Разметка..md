Разрешено ли перестроение?
![[_resources/Pasted image 20260528160810.png]]Разрешено только на соседнюю полосу. Над полосой установлен светофор для реверсивного движения. Над соседней полосой стрелка - значит по этой полосе можно ехать. Над полосой через двойную пунктирную крестик, значит по ней едет встречное движение.

Можно ли выполнить обгон?
![[_resources/Pasted image 20260528164749.png]]
Да, в этом случае можно выполнить обгон. Комбинированная разметка содержит прерывистую полосу со стороны полосы, по которой движется ТС.

Разрешается движение?
![[_resources/Pasted image 20260528164934.png]]
Б - можно, А - нельзя
